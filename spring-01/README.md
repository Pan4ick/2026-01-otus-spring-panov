## Задание 1. Создание простого spring-приложения.

Разработать однопользовательское, консольное приложение для проведения тестирования студентов на основе фиксированного
списка вопросов. Приложение должно запросить фамилию и имя студента, задать вопросы с вариантами ответа и вывести
результат тестирования.

## Требования

1. Весь ввод-вывод осуществляется на английском языке, в т.ч. вопросы и варианты ответов
2. Список вопросов с вариантами ответа хранится в ресурсах приложения, в формате csv
3. Файл с вопросами читается именно как ресурс, а не как файл
4. Все зависимости должны быть настроены в IoC контейнере
5. Контекст описывается XML-файлом
6. Контекст содержит только компоненты приложения и не содержит стандартных классов и доменных объектов
7. Имя ресурса с вопросами (csv-файла) необходимо захардкодить строчкой в XML-файле с контекстом
8. В приложении должна присутствовать объектная модель
9. Все классы в приложении должны решать строго определённую задачу
10. Необходимо написать юнит-тест сервиса тестирования
